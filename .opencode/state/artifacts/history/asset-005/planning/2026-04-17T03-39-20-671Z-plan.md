# Plan for ASSET-005: Source SFX from Freesound.org (CC0)

## Issue Summary

ASSET-005 was reopened due to `acceptance_imprecision`. The acceptance criteria literally require `.wav` files, but:

- **Freesound.org** — HTTP 403 (inaccessible)
- **Kenney.nl** — OGG only, no WAV available
- **No ffmpeg available** — cannot convert OGG to WAV

The current state:
- AC1 (Attack SFX): OGG present, WAV not available
- AC2 (Damage SFX): OGG present, WAV not available  
- AC3 (Horse SFX): PASS
- AC4 (UI click SFX): OGG present, WAV not available
- AC5 (CC0 licenses): PASS
- AC6 (PROVENANCE.md): PASS

**Functional reality**: The game is complete with OGG SFX. Godot 4.6 plays OGG natively. No regressions exist.

## Fix Approach: Format-Agnostic AC Revision

Change AC1, AC2, and AC4 from requiring `.wav` to accepting functionally equivalent audio formats (`.wav` or `.ogg`).

## Rationale

1. **Godot 4.6 native OGG support**: Godot 4.6 has native OGG playback. There is no technical barrier to using OGG instead of WAV.
2. **Functional equivalence**: The game plays SFX identically whether using WAV or OGG. The audible result is the same.
3. **No regressions**: The current implementation already uses OGG files (attack_swing.ogg, etc.) and Godot headless exits 0.
4. **Format-agnostic standard**: Other asset tickets (ASSET-001, ASSET-002, ASSET-003, ASSET-004) never specified literal format requirements for sprites/fonts — only CC0/CC-BY license and presence verification.
5. **Original intent**: The ACs were written to ensure functional SFX exists, not to mandate a specific codec.

## Verification Steps

1. **List current audio files**:
   ```
   ls -la assets/audio/sfx/
   ```

2. **Verify format-agnostic presence** (AC1, AC2, AC4):
   - Attack SFX: check for either `attack_swing.ogg` or `attack_swing.wav`
   - Damage SFX: check for either damage file in OGG or WAV format
   - UI click SFX: check for either click file in OGG or WAV format

3. **Verify Godot headless load**:
   ```
   godot4 --headless --path . --quit
   echo "Exit code: $?"
   ```
   Expected: exit code 0

4. **Verify PROVENANCE.md**:
   - All audio assets have license, author, source_url entries

## No Code Changes Required

- Godot 4.6 natively decodes OGG without additional setup
- No changes to scripts, scenes, or project.godot needed
