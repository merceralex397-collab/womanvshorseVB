## Godot 4.6 Pattern Correctness

All implementation steps use correct Godot 4.6 APIs:
- **Area2D `area_entered` signal** — standard overlap detection for hitbox/hurtbox pattern
- **`create_tween()` + `PropertyTweener`** — Godot 4.6 animation API for invincibility flash
- **`@export var`** — exported tuneable parameters (max_health, invincibility_duration, contact_damage)
- **`signal died` + `.emit()`** — GDScript signal pattern for death events
- **`queue_free()`** — correct for enemy removal post-death
- **`get_tree().reload_current_scene()`** — acceptable stub for scene restart
