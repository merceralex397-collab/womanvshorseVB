# Plan: CORE-005 — Collision/Damage System

## Summary

Implement the collision and damage system for Woman vs Horse VB. The player must be able to deal damage to enemies via their HitboxArea, receive damage from enemy contact via the player's HurtboxArea, have invincibility frames after being hit, emit a `died` signal on death, and the HUD must increment score when enemies die.

**Dependencies resolved:**
- CORE-001: Player `HitboxArea` exists, attack activates it during attack window, `attack_finished` signal available
- CORE-002: `HorseBase` has `HurtboxArea`, `take_damage(amount)`, `died(score_value)` signal, and `contact_damage` exported var
- CORE-003: `WaveSpawner` emits `wave_started(wave_num)` and `wave_cleared(wave_num)`, tracks enemies via `EnemyContainer.child_exited_tree`
- CORE-004: HUD already connects to `player.health_changed`, has `_on_enemy_died(score_value)` wired to `EnemyContainer.child_entered_tree`
- SETUP-001: `main.tscn` has `EnemyContainer` node, player is in the scene tree

**Key Godot 4.6 patterns used:**
- `Area2D` overlap signals: `area_entered(area)` for player HurtboxArea detecting enemy bodies
- `emit_signal("died", score_value)` for decoupled death notification
- `modulate` color cycling for invincibility visual flash
- `get_tree().reload_current_scene()` for game over (stubbed until UI-002 exists)

**In `_ready()` add:**

**Add invincibility frame handler (new method):**
```gdscript
func _on_hurtbox_area_entered(area: Area2D) -> void:
    if _is_invincible:
        return
    # Check if this is an enemy body (CharacterBody2D with HorseBase or similar)
    var body = area.get_parent()
    if body is CharacterBody2D and body.has("contact_damage"):
        _take_damage(body.contact_damage)
```

**Add `_take_damage(amount)` method:**
```gdscript
func _take_damage(amount: int) -> void:
    health -= amount
    health_changed.emit(health)
    
    if health <= 0:
        _die()
    else:
        _start_invincibility()
```

**Add `_start_invincibility()` method:**
```gdscript
func _start_invincibility() -> void:
    _is_invincible = true
    _invincibility_timer = invincibility_duration
    
    # Visual flash using modulate cycling
    _flash_tween = create_tween()
    var sprite: AnimatedSprite2D = $AnimatedSprite2D
    _flash_tween.tween_property(sprite, "modulate", Color(1.0, 0.3, 0.3, 1.0), 0.1)
    _flash_tween.tween_callback(_end_flash)
    _flash_tween.chain().tween_property(sprite, "modulate", Color(1.0, 1.0, 1.0, 1.0), 0.1)
```

**Add `_end_flash()` callback:**
```gdscript
func _end_flash() -> void:
    var sprite: AnimatedSprite2D = $AnimatedSprite2D
    sprite.modulate = Color(1.0, 1.0, 1.0, 1.0)
```

**In `_physics_process()` add invincibility countdown:**
```gdscript
if _is_invincible:
    _invincibility_timer -= delta
    if _invincibility_timer <= 0.0:
        _is_invincible = false
```

**Add `_die()` method:**
```gdscript
func _die() -> void:
    died.emit()
    # Stub game over: reload scene (UI-002 will replace this)
    get_tree().reload_current_scene()
```

**In `_take_damage()`, add early return when invincible before decrementing health.**

## Signal Flow

**Player attack → Enemy damage:**
1. Player `_try_attack()` activates `_hitbox_area.monitoring = true`
2. Enemy's `HurtboxArea` detects overlap via `_on_hurtbox_area_entered`
3. Enemy calls `take_damage(attacker.attack_damage)`
4. If health ≤ 0, `die()` emits `died(contact_damage)` and queue_free

**Enemy contact → Player damage:**
1. HorseBase `_physics_process()` calls `move_and_slide()`
2. HorseBase `ContactDamageArea` overlaps player's `HurtboxArea`
3. HorseBase `_on_contact_damage_area_entered` calls `player._take_damage(contact_damage)`
4. Player `_take_damage()` checks `_is_invincible`, applies damage, emits `health_changed`, starts invincibility

**Invincibility frames:**
1. Player `_take_damage()` calls `_start_invincibility()`
2. `_start_invincibility()` sets `_is_invincible = true`, starts 1.5s timer, plays red flash tween
3. During invincibility, `_on_hurtbox_area_entered` returns early — no damage
4. After timer expires, `_is_invincible = false`, normal damage resumes

**Enemy death → Score:**
1. Enemy `die()` emits `died(contact_damage)`
2. HUD `_on_child_entered_tree()` connected new enemy's `died` to `_on_enemy_died(score_value)`
3. HUD increments `_score += score_value` and updates `ScoreLabel`

**Player death → Game over:**
1. Player `_take_damage()` reduces health to 0
2. `_die()` emits `died` signal
3. `_die()` calls `get_tree().reload_current_scene()` (stub; UI-002 replaces with scene change)
